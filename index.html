EasyDev

Utilities to ease the development of Python packages, including configuration management, logging helpers, and testing tools.

Fitter

A Python package to fit probability distributions to data. Automatically tests all distributions from SciPy, ranks them by goodness-of-fit, and provides simple visualisation of the best fits.

Docs GitHub

Spectrum

A Python package for spectral analysis, providing classical and modern methods for power spectrum estimation including Fourier, Welch, ARMA, and multi-taper approaches.

📄 Publications

Sequana Coverage: Detection of Genomic Variations with a DNA Coverage Analysis Tool

GigaScience, 2018

Presents a standalone tool for analysing DNA coverage data from sequencing experiments. The method detects low and high coverage regions, identifies outliers, and generates HTML reports for quality control.

Sequana: a set of Snakemake NGS pipelines

Journal of Open Source Software, 2017

Introduces the Sequana framework providing a collection of modular, reproducible NGS pipelines built on Snakemake. Covers RNA-seq, variant calling, quality control, and de-novo assembly workflows.

BioServices: a common Python package to access biological web services

Bioinformatics, 2013

Describes BioServices, a Python package providing a unified interface to many biological databases and web services. Removes the need to manage heterogeneous APIs and simplifies bioinformatics scripting workflows.

Training Signalling Pathway Maps to Biochemical Data with Constrained Fuzzy Logic

PLOS Computational Biology, 2012

Introduces a constrained fuzzy logic formalism to model signal transduction networks, enabling inference of pathway activity from phosphoproteomic data. Implemented in the open-source CellNOpt toolbox.

CellNOptR: a flexible toolkit to train protein signalling networks to data using multiple logic formalisms

PLOS ONE, 2012

Presents CellNOptR, an R package for training Boolean and fuzzy logic models of signalling networks to phosphoproteomic data. Supports multiple logic formalisms and integrates with the broader CellNOpt toolbox ecosystem.

CNOFuzzy: a flexible toolkit to train fuzzy logic models to data

Bioinformatics, 2010

Describes the CNOFuzzy toolbox for training fuzzy logic models of signal transduction to experimental data, extending Boolean modelling to handle graded biological responses and continuous measurements.

CellNOpt: using signalling pathway models to predict dynamic drug-combination effects

BMC Systems Biology, 2009

Introduces the CellNOpt framework for training prior knowledge signalling network models to experimental data, enabling prediction of drug-combination effects and identification of network rewiring in disease contexts.
